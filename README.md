# How it work

- Sign in form
  - Not implement code sign in
  - Route to Sign in (default route)
  - If not yet have account click "Sign up an account", route to Sign up.
  - Click Sign in button to route to dashboard

- Sign up from
  - Not implement code sign up
  - Click "Already have an account" or Sign up button to route to Sign in.

- Navbar
  - Not implement code logout
  - Click Logout route to Sign in form

- Dashboard
  - Show summarize information like New Orders, User Registrations,...etc.

- Profile
  - Show information of profile
